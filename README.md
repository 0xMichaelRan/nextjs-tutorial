## Next.js App Router Course - Starter

```
npx create-next-app@latest nextjs-dashboard --example "https://github.com/vercel/next-learn/tree/main/dashboard/starter-example" --use-pnpm
cd nextjs-dashboard
git status
pnpm i
pnpm dev
```
